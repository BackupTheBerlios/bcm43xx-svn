Broadcom 43xx Linux Driver

An attempt to write a Broadcom 43xx driver for Linux, based on this specification: http://bcm-specs.sipsolutions.net/

Access the SVN repository and join the development mailing list

Overview

A driver for the Broadcom 43xx wireless chip.

This can be found in many devices, including (but not limited to!): AirPort Extreme, TrueMobile 1300 WLAN Mini-PCI Card, Wireless 1350 WLAN Mini-PCI Card, WL-100G, Linksys WMP54G PCI, Belkin F5D7010 54g Wireless Network card, etc.

Broadcom never released specification about these chips.

You can find more details about the specification we are using at http://linux-bcom4301.sourceforge.net/, more general information here.
